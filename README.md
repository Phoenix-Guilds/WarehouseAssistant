# WarehouseAssistant (Sirus x5 — Phoenix Nest)

Специализированный аддон для координации работы складских аккаунтов гильдии **Phoenix Nest**.

## Назначение
Аддон предназначен в основном только для **помощников по складу**, имеющих доступ к общим гильдейским аккаунтам. Он позволяет в реальном времени видеть, какие из 5 общих аккаунтов сейчас заняты другими участниками, чтобы избежать конфликтов при входе.

> **Важно:** Если Вы состоите в гильдии **Phoenix Nest**, но у вас нет доступа к общим аккаунтам, данный аддон вам не требуется, но Вы можете его у себя установить, чтобы видеть, когда на складе кто-то из помощников есть.

## Функционал
* **Мониторинг аккаунтов (1-5):**
    * `1–4` — Склады (выдача предметов).
    * `5` — Банкиры (прием предметов).
* **Индикация занятости:** Если кто-то зашел на персонажа конкретного аккаунта, номер этого аккаунта подсвечивается **красным** (задержка обновления ~5 сек).
* **Списки персонажей:** При наведении мыши на номер отображается список всех чаров на аккаунте. Никнейм текущего складского персонажа онлайн подсвечивается **зеленым**.
* **Интерфейс:**
    * Свободное перемещение фрейма по экрану.
    * Кнопка сворачивания в узкую полоску (в верхнем правом углу фрейма).

## Установка

1.  Перейдите в раздел [Releases](https://github.com/WorldOfSitesNet/WarehouseAssistant/releases) (справа).
2.  В самом последнем релизе скачайте архив `WarehouseAssistant.zip`.
3.  Распакуйте содержимое архива.
4.  Если Вы уже пользовались или используете этот аддон, то удалите из папки с аддонами `World of Warcraft Sirus\Interface\AddOns\` папку `WarehouseAssistant` и приступайте к следующему пункту.
5.  Переместите распакованную папку `WarehouseAssistant` в директорию с игрой по пути:
    `World of Warcraft Sirus\Interface\AddOns\`.
5.  Выполните вход в игру и убедитесь, что в меню выбора персонажей, в разделе **"Модификации"**, аддон включен.
